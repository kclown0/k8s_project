# k8s_project

## 프로젝트 설명
- 쿠베네티스를 활용한 공항데이터 mlops 파이프라인 구현
- 항공 정보를 입력하면 지연시간을 예측하는 ML서비스를 제공하는 프로젝트
- 기존 Docker 환경에서 쿠버네티스를 사용하는 환경으로 업그레이드 진행
## 프로젝트 기간
